Movement Map emblem cards (square, ~1080x1080 or larger, PNG).
Save the six type emblems here with these EXACT filenames:

  gripper.png      (The Gripper — Type 01)
  rangeChaser.png  (The Range Chaser — Type 02)
  hiker.png        (The Hiker — Type 03)
  twister.png      (The Twister — Type 04)
  collapser.png    (The Collapser — Type 05)
  controller.png   (The Controller — Type 06)

The result page shows /movement-map/<type>.png as the shareable card.
If a file is missing, the app falls back to a plain branded card automatically.
